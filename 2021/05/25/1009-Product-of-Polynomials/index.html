1009_Product_of_Polynomials

Author: Arost Date: May 25, 2021  9:38:07 Category: PTA

This time, you are supposed to find A×B where A and B are two polynomials.

Input Specification:

Each input file contains one test case. Each case occupies 2 lines, and each line contains the information of a polynomial:

K N1 aN1 N2 aN2 … NK aNK

where K is the number of nonzero terms in the polynomial, N**i and aNi (i=1,2,⋯,K) are the exponents and coefficients, respectively. It is given that 1≤K≤10, 0≤N**K<⋯<N2<N1≤1000.

Output Specification:

For each test case you should output the product of A and B in one line, with the same format as the input. Notice that there must be NO extra space at the end of each line. Please be accurate up to 1 decimal place.

题目大意：

进制是二进制！！！所以基数一律为2

第一行表示数A，第二行表示数B，输出A*B后的值。

数A有两项，第一项指数是1，系数是2.4，即$2.4 * 2^1$；第二项指数是0，系数是3.2，即$3.2 * 2^0$。
$$
A = 2.4 * 2^1 + 3.2 * 2^0 = 8
$$
数A有两项，第一项指数是2，系数是1.5，即$1.5 * 2^2$；第二项指数是1，系数是0.5，即$1.5 * 2^0$。
$$
B = 1.5 * 2^2 + 0.5 * 2^1 = 7
$$
结果值就是
$$
7 * 8 = 56 = 3.6 * 2^3 + 6.0 * 2^2 + 1.6 * 2^1
$$

思路：

A和B的每个项都”相乘“

$2.4 * 1.5 = 3.6$的结果就是结果式中的一项，该项的的系数就是3.6，该项的指数就是$1+2$，其中1是A中2.4那一项的指数，2是B中1.5对应的指数。所以结果式中就产生了一项 $3.6 * 2^3$ 。

$2.4 * 0.5 = 1.2$的结果就是结果式中的一项，该项的的系数就是1.2，该项的指数就是$1+1$，其中1是A中2.4那一项的指数，第二个1是B中0.5对应的指数。所以结果式中就产生了一项$1.2 * 2^2$。

$3.2 * 1.5 = 4.8$的结果就是结果式中的一项，该项的的系数就是4.8，该项的指数就是$0+2$，其中0是A中3.2那一项的指数，2是B中1.5对应的指数。所以结果式中就产生了一项$4.8 * 2^2$。

$3.2 * 0.5 = 1.6$的结果就是结果式中的一项，该项的的系数就是1.6，该项的指数就是$0+1$，其中0是A中3.2那一项的指数，1是B中0.5对应的指数。所以结果式中就产生了一项$1.6 * 2^1$。

注意到$1.2 * 2^2$和$4.8 * 2^2$的指数都是2，那么就需要把系数相加，最后产生的项为$6.0 * 2^2$

所以我们可以设计一个数组，数组的下标刚好表示当前项的指数，该下标对应的值表示该项的系数。
