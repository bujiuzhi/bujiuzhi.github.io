1001_A+B_Format

Author: Arost Date: May 13, 2021  21:27:41 Category: PTA

Calculate a+b and output the sum in standard format – that is, the digits must be separated into groups of three by commas (unless there are less than four digits).

Input Specification:

Each input file contains one test case. Each case contains a pair of integers a and b where −106≤a,b≤106. The numbers are separated by a space.

Output Specification:

For each test case, you should output the sum of a and b in one line. The sum must be written in the standard format.

题目大意：

输出两个数的和，要求加逗号。

思路：

例如，-1,999,398,341,341中surplus=2，指的是最前面的那一组元素个数。

i+1可以理解为当前第几个。

遇到第一个逗号时，1的下标i是1,那么i+1=2，即第一个逗号前面有2个元素；

遇到第二个逗号时，9的下标是4，那么i+1=5，即第二个逗号前面有5个元素；

遇到第三个逗号时，8的下标是7，那么i+1=8，即第三个逗号前面有8个元素；

遇到第四个逗号时，1的下标是10，那么i+1=11，即第四个逗号前面有11个元素；

我们发现，后面跟这逗号的条件是当前元素下标从2开始不停的+3，也就是说，当前元素的个数%3=最前面多出来的那部分

当前元素的个数，即i+1;

最前面多出来的那部分元素个数，即len%3;

即(i + 1) % 3 == surplus，同时注意最后一个不用加逗号。
