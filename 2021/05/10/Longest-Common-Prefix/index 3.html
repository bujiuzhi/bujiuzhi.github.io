14. 最长公共前缀

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。

提示：

  • 0 <= strs.length <= 200
  • 0 <= strs[i].length <= 200
  • strs[i] 仅由小写英文字母组成

思路：

先记录第一个字符串s = strs[0],从第二个字符串开始和s比，如果有不一样的地方，那么就需要更新s（从不一样的下标开始，后面的都不要了——s = s.substring(0, k)），这时就可以用新的s和下一个字符串比了。在{"flower", "flow", "flight"}案例中,s先等于"flower" ，和"flow"比较，之后更新 s为"flow"，然后再用新的s和"flight"做比较，最后得到"fl"。

当"flower"和"flow"比较,两个字符串并不是因为中间字符的不一样而更新的s，是比较之后检验下标发现的，这是我们就需要再次做一次更新。
